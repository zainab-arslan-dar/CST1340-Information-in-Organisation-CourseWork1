# Bookshop Website — XML/XSL Web Development Project
# Overview
This project is a responsive bookshop website developed as part of the CST1340 Web Development module. It demonstrates the use of structured data technologies alongside front-end development to separate content, structure, and presentation.
The application uses XML to store book data, XSLT to transform and display the data, and DTD to validate the structure, combined with HTML and CSS for the user interface.

# Features
Responsive user interface implemented using HTML5 and CSS3
Book catalogue structured and stored in XML format
Data transformation using XSLT for rendering content in the browser
XML validation through DTD to ensure data consistency
Separation of data, logic, and presentation
Technologies Used
HTML5
CSS3
XML
XSLT
DTD (Document Type Definition)

# How It Works

The book data is stored in an XML file, which follows a predefined structure enforced by a DTD. The XSLT file transforms this XML data into HTML, allowing it to be displayed in a browser-friendly format. CSS is applied to enhance the layout and ensure responsiveness across devices.
